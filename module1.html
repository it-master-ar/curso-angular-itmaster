Controlador

Intermediario, interactua con el modelo en respuesta a interacciones del usuario (Vista).

Ventajas

  • Separación de conceptos.
  • Bajo nivel de acoplamiento (no side effects).
  • Facilita el testing.
  • Facilita la transferencia tecnologia.

Angular MVC / MV*

  • Modelo: representación de datos, interación con el servidor y persistencia.
  • Vista: visualización e interacción con los datos (GUI).
  • Controlador (VM): expone el modelo a la vista, reponde interacciones de usuario y maneja estados visuales.

Vista

Markup (HTML), filtros y expresiones (simples)

Modelo

JS (servicios), persistencia y lógica de negocio.

Controller (VM)

Eventos y vinculación de datos.

NO manipula el DOM
