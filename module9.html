Routing

  • Redireccionamiento client side
  • Mapea controladores con vistas a traves de URL's
  • Prefijo # (se puede evitar usando modo HTML5)
  • Intercepta URL's (posterior a la carga del index)
  • Modulo: ngRoute

Componentes

  • Directiva ng-view (container).
  • Servicio $routeProvider, $route y $routeParams.
  • .when(): define un endopoint
  • .otherwise(): define endopoint general

Sintaxis

          
            angular.module('OWMApp', ['ngRoute'])
              .config(['$routeProvider', function($routeProvider){
                  $routeProvider
                    .when('/', {
                      templateUrl : 'home.html',
                      controller : 'HomeCtrl'
                    })
                    .when('/city/:name', {
                      templateUrl : 'city.html',
                      controller : 'CityCtrl'
                    })
                    .otherwise('/error');
              }]);

Propiedades

  • templateUrl
  • controller
  • resolve

.config()

  • Previos al metodo .run()
  • Solo pueden recibir providers y constants.
  • Previo a la declaracion de services (y factories).

$location

  • Version angular de window.location
  • La URL del browser es mapeada con este servicio
