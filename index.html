Why Auth0?

Single sign-on system with easy and quick set up.

"Auth0 eliminates the friction of authenticacion in apps and APIs."

  • Choose providers
  • Login page customization
  • Analytics and users stats
  • Developer oriented

Apps

Connect any app with any device (and any program language).

  • Multiple SDK's
  • Login Box Integration
  • Seed projects to setup easy

Users

GUI to administrate application users.

  • Add custom attributes
  • Get user history
  • Handle user profiles

Connectcions

Middlewares to get and connect users with your apps

  • Social (Facebook, Twitter, etc)
  • Enterprise (AD, LDAP, SAML)
  • Database (Auth0)
  • Passwordless (TouchID, Twilio)

Rules

Set of code that will be executed as part of the authenticacion pipeline every time a user authenticates to an app.

Emails

Emails send system (with template customization) to verify users, change passwords, welcome users, etc.

Addons

Extensions associated with an application. 3rd party APIs (Salesforce, Azure Service Bus, Azure Mobile Services, SAP, etc).

MFA

Multi-Factor authentication via Google Authenticator and Duo.

You can trigger MFA base on conditions as user location, ip filtering, device, etc.

JWT

Open standard RFC 7519 method for representing claims securely between two parties.
